\Base2

\Base2QuickMail

Quick setup and wrapping to send mails.

This class wraps Zend\Mail in order to configure and send mails as quickly and simple as possible according to our most common needs when sending mail.

By default this means sending an HTML mail via Sendmail, SMTP or InMemory.

It depends on a proper configuration defined in local.php (or maybe partially in global.php) to work. See config/quickmail.config.php.dist for a sample configuration with explanations.

You can get access to the Zend\Mail\Message and Zend\Mail\Transport objects if you need to handle a special case.
